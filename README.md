# InkuManga

**InkuManga** is a web-based manga discovery platform built with **React.js** and **Tailwind CSS**. It offers manga lovers a fast, responsive, and engaging experience to explore trending, top-rated, and personalized manga recommendations.

🔗 **Live Demo**: [https://inkumanga.netlify.app/](https://inkumanga.netlify.app/)

---

## 🌟 Features

- 🔍 **Search & Discover**: Quickly find manga by title, genre, or ranking.
- 📊 **Trending Titles**: Browse the most popular and up-to-date manga series.
- ❤️ **Favorites System**: Bookmark and save your favorite manga titles (client-side).
- 🎨 **Responsive UI**: Optimized for mobile and desktop with Tailwind CSS.
- ⚡ **Fast Performance**: Built with React for quick rendering and navigation.

---

## 🛠️ Technologies Used

- **React.js** – For building a modular, component-based frontend.
- **Tailwind CSS** – For fast, utility-first styling.
- **React Router DOM** – To handle SPA routing.
- **Netlify** – For seamless deployment and hosting.
